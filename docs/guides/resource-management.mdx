# Resource Management Guide

Efficient resource management is the key to long-term success in Last Asylum.

## Types of Resources

### Premium Currency
The most valuable resource - spend wisely!

**How to Earn:**
- Daily/Weekly missions
- Arena rewards
- Achievements
- Special events
- Abyss/Tower clears

**Best Uses:**
1. Limited character banners
2. Energy refills during events
3. Essential progression items

### Gold/Credits

Used for nearly everything:
- Upgrading characters
- Enhancing gear
- Shop purchases
- Skill upgrades

**Farming Methods:**
- Gold dungeons (weekdays)
- Selling excess gear
- Daily missions
- Guild activities

### Stamina/Energy

Your limiting factor for progression.

| Source | Amount | Refresh |
|--------|--------|---------|
| Natural Regen | 1/5 min | Constant |
| Daily Free | 60-120 | Reset daily |
| Friend Send | 20-40 | Daily |
|Mailbox | Varies | Events |

## Farming Priorities

### Early Game (Week 1-2)
```
Priority: Campaign Progress > Character EXP > Basic Gear
```

### Mid Game (Month 1-2)
```
Priority: Gear Farming > Skill Materials > Awakening Items
```

### Late Game (Month 3+)
```
Priority: Min-maxing Gear > New Characters > Collectibles
```

## Weekly Schedule

Optimize your farming with this schedule:

| Day | Focus | Why |
|-----|-------|-----|
| Monday | EXP Dungeon | Level up new characters |
| Tuesday | Gold Dungeon | Upgrade funds |
| Wednesday | Skill Materials | Power up abilities |
| Thursday | Gold Dungeon | More upgrades |
| Friday | Gear Dungeon | Equipment farming |
| Saturday | Mixed/Events | Catch up or events |
| Sunday | Mixed/Events | Preparation |

## Resource Sinks to Avoid

:::danger[Don't Waste Resources On:]

- Upgrading every character equally
- Refreshing shops constantly
- Low-value gear enhancements
- Non-essential cosmetics (early game)

:::

## Event Efficiency

Events often provide the best value:

1. **Clear event shop** priorities:
   - Limited characters/items
   - Summoning currency
   - Rare materials
   - Skip: Common materials you can farm

2. **Event energy** usage:
   - Use ALL bonus stages
   - Farm highest difficulty you can clear
   - Buy energy refills if good value

## Quick Tips

- 💡 Never cap on energy - always be farming
- 💡 Stack resources before upgrade events  
- 💡 Guild donations = free rewards
- 💡 Check shop refreshes for deals
- 💡 Sell duplicate gear for gold
