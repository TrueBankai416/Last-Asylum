## Rerun Predictions

### Likely to Return
Events that typically rerun:
- Holiday events (annually)
- Anniversary events
- Popular collaboration events

### Probably One-Time
Events unlikely to return:
- Launch celebration
- Milestone events
- Time-specific collaborations

## Why Track Past Events?

1. **Predict reruns** - Know what might come back
2. **Budget planning** - Estimate resource needs
3. **Team preparation** - Build for expected content
4. **Reward comparison** - See reward value trends
