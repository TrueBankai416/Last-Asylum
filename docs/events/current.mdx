## How Events Work

### Event Types

| Type | Description | Priority |
|------|-------------|----------|
| **Limited Banner** | Special character summons | High for target character |
| **Story Event** | Special stages with rewards | High - unique rewards |
| **Raid Event** | Cooperative boss fights | High - limited time |
| **Collection Event** | Gather items for rewards | Medium - passive |
| **Login Event** | Daily login bonuses | Low - just log in |

### Event Priorities

When multiple events run:
```
Priority Order:
1. Limited banners (if you want the character)
2. Time-limited unique rewards
3. High-value farmable events
4. Regular play activities
```
