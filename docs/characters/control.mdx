# Control Characters 🎯

Controllers dominate the battlefield through crowd control, debuffs, and disruption. They prevent enemies from acting at full effectiveness.

## What Makes a Good Controller?

**Primary Stats:**
- High Speed (control first!)
- Effectiveness (land debuffs)
- HP/DEF (survive)

**Key Abilities:**
- Stun, Freeze, Sleep (prevent turns)
- Silence (prevent skills)
- Provoke (redirect attacks)
- Strip (remove buffs)

## Control Subtypes

### Hard CC
Completely prevent enemy actions.
- **Abilities:** Stun, Freeze, Petrify
- **Best for:** Buying time, interrupt
- **Weakness:** Immunity, Resistance

### Soft CC
Limit enemy effectiveness.
- **Abilities:** Silence, Blind, Slow
- **Best for:** Specific counters
- **Weakness:** Less impactful than hard CC

### Debuffers
Weaken enemies directly.
- **Abilities:** Defense down, Attack down
- **Best for:** Increasing damage, reducing threat
- **Weakness:** Cleanse counters

### Strippers
Remove enemy buffs.
- **Abilities:** Dispel, Buff block
- **Best for:** Counter buff teams
- **Weakness:** Need follow-up

## Control Gear Guide

**Main Stats Priority:**
1. Speed on boots (go first!)
2. Effectiveness% on ring
3. HP% for survival

**Substats Priority:**
- Speed > Effectiveness > HP% > DEF%

**Effectiveness Requirements:**
```
Content            | Effectiveness Needed
-------------------|--------------------
Normal Content     | 55%
Hard Content       | 85%
Boss Fights        | 100%+
PvP               | 100%+ (varies)
```

**Recommended Sets:**
- Speed Set (absolutely essential)
- Hit/Effectiveness Set
- HP Set (survival)

## Control Strategies

### Chain Control
```
Character A → Stun enemy
Character B → Extend stun / add more CC
Enemy → Never gets a turn
```

### Strip + Control
```
Character A → Remove immunity
Character B → Apply stun
Enemy → Controlled despite immunity
```

:::warning[Resistance Check]

Your CC can miss! Stack effectiveness to overcome enemy resistance.
:::
